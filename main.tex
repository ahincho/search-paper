% ---- Abstract ----
\begin{abstract}
El algoritmo A* constituye uno de los métodos de búsqueda heurística más ampliamente utilizados en inteligencia artificial para la resolución de problemas de búsqueda de caminos óptimos. Este trabajo presenta el diseño, implementación y validación de una librería extensible en Python denominada \texttt{search-library}, que implementa el algoritmo A* aplicado tanto a grafos ponderados como a matrices bidimensionales (grids). Se realiza un análisis comparativo exhaustivo con los algoritmos BFS, DFS y Dijkstra, evaluando complejidad temporal, complejidad espacial y número de nodos explorados. Los resultados experimentales demuestran que A* reduce significativamente el espacio de búsqueda respecto a Dijkstra (hasta un 62\% menos de nodos explorados) mientras mantiene la garantía de optimalidad cuando se emplea una heurística admisible. La arquitectura basada en patrones de diseño Strategy y Adapter permite la extensibilidad del framework para diversos dominios de aplicación, incluyendo videojuegos, robótica y sistemas de navegación.
\end{abstract}
